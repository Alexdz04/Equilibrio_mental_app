Equilibrio mental - App movil

Descripción del proyecto:

Equilibrio mental, nuestro proyecto es una plataforma digital de salud mental diseñada para democratizar el acceso a servicios psicológicos en el sector público de RD. Este prototipo funcional fue desarrollado específicamente para el Hospital General Dr. Vinicio Calventi, buscando eliminar barreras de acceso como costos, distancia y estigma social.

La aplicacion permite a los pacientes gestionar su bienestar emocional desde su dispositivo móvil, facilitando la conexión remota con especialistas y el acceso a herramientas de autoayuda.

Funcionalidades principales:

Autenticación segura: Registro e inicio de sesión para pacientes y personal médico.

Gestión de citas: Agenda interactiva para reservar consultas presenciales o virtuales.

Telepsicología: Módulo de videollamadas integrado para sesiones de terapia remota.

Recursos de bienestar: Biblioteca multimedia con artículos, audios y guías para el manejo de ansiedad y estrés.

Directorio médico: Perfiles detallados de psicólogos y psiquiatras con valoraciones.

Tecnologías utilizadas:

Frontend: Flutter.

Lenguaje: Dart.

Librerías clave:

table_calendar: Gestión de calendario y agenda.

google_fonts: Tipografía profesional (Lato).

intl: Formato y localización de fechas.

Instalación y ejecución:

Para ejecutar este proyecto localmente, hay que clonar el repositorio:

git clone [https://github.com/Alexdz04/Equilibrio_mental_app.git](https://github.com/Alexdz04/Equilibrio_mental_app.git)


Instalar dependencias:

cd equilibrio_mental
flutter pub get


Ejecutar la app:

flutter run
